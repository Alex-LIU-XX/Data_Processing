## 5. 数据特点与潜在问题

### 5.1 夹爪状态分析

- 全开（<0.01）: 3,242 帧 (26.6%)
- 全闭（>0.6）: 1,752 帧 (14.3%)
- 过渡态: 7,215 帧 (59.1%)

**特点**: 夹爪不是简单的二值开/关，大量帧处于中间过渡状态，为比例控制模式。

### 5.2 时间戳异常

采样间隔统计: **均值 1.0ms, 标准差 1,099ms**

这说明时间戳并非严格均匀的 100ms（10fps），可能存在时间戳跳跃或记录问题，需要进一步调查。这可能是 S2 阶段需要关注的问题。

### 5.3 Action = Next State

action 本质上就是 state_{t+1}（99.19% 的帧完全匹配），意味着：
- Action 空间 = State 空间（绝对关节位置）
- delta = state_{t+1} - state_t 是隐式的运动信息
- 最后一个帧的 action 使用 state_{last} 进行 tail padding（占 0.81% 的不匹配帧）

### 5.4 数据质量评估

| 检查项 | 结果 | 风险 |
|--------|------|------|
| S1 突变 | 少量，需进一步联合阈值判断 | 低 |
| S2 对齐 | DA > 97%，质量高 | 低 |
| S3 极值 | 无异常 | 无 |
| S4 FK | 需 URDF 验证 | 未知 |
| S5 坐标系 | 单数据集，无需对齐 | 无 |
| C1 指令一致性 | 42 个同义指令，需 VLM 验证 | 中（语义变体可能引入歧义） |
| C2 视频-状态一致性 | 需渲染验证 | 未知 |
| C3 视频质量 | 需检测视频文件 | 未知 |

**总体评价**: 此数据集质量较高，S1-S3 阶段的信号完整性良好。主要潜在问题在于时间戳不均匀（可能因视频帧率波动）和夹爪的连续控制模式。如果复现完整管线，建议优先验证 C2 视频-状态一致性和 C1 指令语义一致性。
